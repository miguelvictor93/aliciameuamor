# **App Name**: Qual Princesa É Essa?

## Core Features:

- Quiz Progression System: Manages the game's flow through the start screen, interactive quiz questions, immediate feedback, and the final score screen.
- Interactive Question Display: Presents each princess's image within a golden medieval frame, displays the question text, and offers four selectable answer buttons.
- Real-time Answer Feedback: Provides immediate visual feedback for correct or incorrect answers, including button styling, confetti for correct, shaking for incorrect, and appropriate messages.
- Scoring and Game State Management: Tracks the player's score, current question number, and manages the game's overall state, including displaying a golden progress bar.
- Dynamic Content Shuffling: Randomizes the order of quiz questions and their corresponding answer options at the start of each new game for varied gameplay.
- Animated Enchanted Background: Displays various animated background elements, including slowly moving semi-transparent clouds, a pulsating rainbow, and twinkling particles to create an immersive fairytale ambiance.
- AI-Powered Quiz Question Augmentation Tool: Utilizes a generative AI tool to create additional diverse quiz questions and corresponding answer options about various princesses, expanding the game's content pool.

## Style Guidelines:

- Primary action and main text color: Dark Purplish Pink (#7B3F6E) to evoke royalty and fantasy against the light background.
- Background gradient: Soft transitions from a very light pinkish purple (approx. HSL(297, 15%, 95%) or #F6EDF8 for a general base hue) through user-specified #ffe4f0, #ffd6f0, to #e8d5ff for an enchanted sky effect.
- Accent and highlight elements: Rich golds such as #FFD700, #DAA520, and #FFF0A0 are used for decorative frame borders, text shimmer, and call-to-action buttons.
- Feedback colors: Light green (#A8E6A3) for correct answers and soft red (#FFB3B3) for incorrect answers. Normal buttons use a very light pink (#FFF0F8) with a golden border.
- Headline and title font: 'Cinzel Decorative' (serif) for a majestic, medieval fantasy aesthetic. Note: currently only Google Fonts are supported.
- Body and button text font: 'Quicksand' (sans-serif) for a rounded, cute, and friendly feel, suitable for interactive elements. Note: currently only Google Fonts are supported.
- Utilize various star (⭐) and sparkle (✨) emojis or similar glyphs for background particles, correct answer feedback, and general enchantment. A crown emoji (👑) serves as the custom cursor.
- Responsive, mobile-first design supporting screens from 320px up to 1200px. Mobile layouts feature smaller images (220px × 260px) and stacked answer buttons. Desktop layouts are centrally aligned with a maximum width of 600px.
- The game structure includes an animated title start screen, a central quiz screen with image, question, and answer buttons, and a final score screen displaying points, star ratings, and a restart option. A golden progress bar is consistently displayed at the top during gameplay.
- Background animations include clouds slowly moving left-to-right (20-40s duration), a semi-transparent rainbow pulsating softly (scale 1.0 to 1.05), and small twinkling star particles (opacity/scale variations).
- Interactive elements feature smooth transitions: images fade in and scale up (0.8 to 1.0 in 0.6s) upon entry. Correct buttons pulse with a green glow, while incorrect buttons shake and turn red.
- Feedback animations include colorful confetti showers and ascending star visuals for correct answers. The main title shimmers with an animated golden gradient, and buttons exhibit subtle elevation and rosada shadow on hover. Transitions between questions use a smooth fadeOut then fadeIn effect.
